Game Programming

Making video games is something quite a lot of people dream of. But since you're here, you can probably start making that dream real. Where to start learning Game Programming depends on how much prior programming knowledge you have. The list of resources assumes you start from 0. If not, feel free to look further down the list for more advanced resources.

Python

I recommend picking up Python as a first programming language, it's simple to understand yet is extremely useful. Python will also teach you how to program without having to worry too much about syntax and/or other quirks of more complex programming languages.


Resources:

  1. Python Documentation Tutorial
  2. Learn Python the Hard Way

Visual Basic

Despite Visual Basic being a notoriously annoying language, it doesn't change the fact that it truly excels at making small video games. Compared to other computer languages, making small games is truly a breeze in Visual Basic.

Resources:

  1. Learning Visual Basic .NET
  2. Microsoft Documentation

C

C is a more advanced programming language. It's almost 50 years old but it's the father of a lot of other modern programming languages. It's also an excellent language to learn more advanced programming concepts such as memory management. Learning C will certainly give you the necessary skills to learn new languages much more easily.


Resources:

  1. Programming in C
  2. The C Programming Language

Object Oriented Programming

Object Oriented Programming is a kind of programming style that's used pretty much everywhere. It is a vital skill to learn for anybody that's interested in software engingeering. One can learn OOP in a variety of languages, Java, C++, Ruby to name a few.


Resources:

  2. C++
