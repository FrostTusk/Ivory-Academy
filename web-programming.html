Web Programming

Web technology is very popular nowadays. It's also very attractive to novice programmers because it's fairly easy to start building your own websites. In fact, I built this entire website after only doing the introductory HTML/CSS sessions of Codeacademy. The following list of resources assumes you start from 0. If not, feel free to look further down the list for more advanced resources.

HTML/CSS

HTML and CSS are the base of how a website is built, using HTML/CSS you can build your own static websites, such as this one. HTML/CSS is not really a programming language yet they're vital skills to have if you want to do any serious web programming.


Resources:

  1. HTML and CSS: Design and Build Websites
  2. Codeacademy

JavaScript/JQuery

With HTML/CSS you can build a website, with JavaScript/JQuery you can build an experience. Learning JavaScript will not only allow you make beautiful, dynamic websites, it'll also teach you basic programming. This will help you develop other applications in the future that are not web based.


Resources:

  1. JavaScript and JQuery: Interactive Front-End Web Development

Python

Python is an incredibly useful scripting language. It's simple to understand yet is extremely useful. It is especially useful for web programming due to the incredibly popular django/rest framework. Using this framework you can build a backend for your website.


Resources:

  1. Python Documentation Tutorial
  2. Learn Python the Hard Way
  3. Django REST framework
